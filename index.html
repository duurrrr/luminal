Setup

LUMINAL runs AI models locally on your machine — nothing is sent to the cloud.
Download the script for your role, install the dependencies once, and run it before opening the site.

Streamers — Step 1 of 2
BLIP SERVER
The vision AI that runs on your machine and describes your camera feed. Stays 100% local — nothing is sent to the cloud.
Required by: Streamers only
↓ blip_server.py
Everyone — Step 2 of 2
NGROK
A free tunnel that lets the website securely reach your local AI server. Required for both streamers (port 5555) and viewers (port 5556) because the site runs on HTTPS. Free account, one-time setup.
Required by: Streamers & Viewers
↗ ngrok.com/download
Viewers — Step 1 of 2
SDXL SERVER
Runs image generation locally on your machine. Reconstructs the scene from AI descriptions using Stable Diffusion Turbo.
Required by: Viewers only
↓ sdxl_turbo_server.py
DEPENDENCIES — run once in terminal
pip install "diffusers==0.27.2" "transformers==4.40.0" "huggingface_hub==0.23.4" "accelerate==0.27.2" torch torchvision pillow flask flask-cors numpy
pip uninstall xformers peft -y
If you are streaming
1
Download blip_server.py from the Downloads tab and install the dependencies listed there (one time only).
2
Open a terminal and run: python blip_server.py — wait until it says BLIP model loaded. Keep this terminal open.
3
Download and install ngrok from the Downloads tab (one time only). Create a free account at ngrok.com, then run: ngrok config add-authtoken YOUR_TOKEN with the token from your ngrok dashboard.
4
Open a second terminal and run: ngrok http 5555 — ngrok will show a URL like https://abc123.ngrok-free.app. Copy that URL. Keep this terminal open too.
5
Come back here and click STREAM. Paste your ngrok URL into the BLIP URL field, allow camera access, then click GO LIVE.
6
Share the 4-letter stream key shown on screen with whoever wants to watch. They just need this page — no downloads required.
If you are watching
1
Download sdxl_turbo_server.py from the Downloads tab and install the dependencies listed there (one time only). The first run downloads ~2GB.
2
Open a terminal and run: python sdxl_turbo_server.py — wait until it says SD Turbo ready. Keep this terminal open.
3
Download and install ngrok (one time only). Set up your free account, then run: ngrok http 5556 — copy the https://xxxx.ngrok-free.app URL it gives you. Keep this terminal open too.
4
Come back here and click WATCH. Paste your ngrok URL into the SDXL URL field, enter the stream key from the streamer, and click JOIN.
5
The first image takes a moment to generate. After that, images update continuously as the AI reconstructs what the camera sees.
GPU recommended
!
Both scripts run on CPU but are much faster with an Nvidia GPU. If you have one, make sure you have the CUDA version of PyTorch: pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
!
Blank images on the viewer side usually mean a float16 precision issue. Your GPU is still supported — the server handles this automatically.
